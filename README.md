# Project Overview
Customer churn is a critical challenge for service-based businesses. This project provides a deep dive into customer behavior, identifying the primary factors contributing to attrition. By analyzing patterns in tenure, contract types, and service usage, this project transforms raw data into actionable business intelligence to improve retention and revenue.

 Database Management/Analysis: SQL
 Visualization & BI: Power BI / Tableau

# Key Insights & Findings
 High-Risk Segments: Identified that customers on "Month-to-Month" contracts exhibit significantly higher churn rates.
 Support Impact: A strong correlation was found between the volume of support tickets and the likelihood of cancellation.
 Revenue Drivers: Long-term tenure customers contribute significantly more to the Lifetime Value (CLV) of the company.
 
# Strategic Recommendations
1. Incentivize Long-term Contracts: Offer discounts for customers transitioning from monthly to yearly plans.
2. Proactive Support: Implement a "priority flag" for customers with high support ticket counts to address issues before they churn.
3. Targeted Retention: Create personalized loyalty programs for high-value customers at the end of their contract cycles.
